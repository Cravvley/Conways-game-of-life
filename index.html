Conway's Game of Life

How to play

Start: Set grid size (3–65), optionally check Random fill, click Start.

Controls: next — advance; auto — run; stop — pause. Step = generations per click.

Edit: Click cells to toggle alive/dead (when paused).

Rules: Dead + 3 neighbors → alive. Alive + 2 or 3 → stays. Otherwise → dies.
